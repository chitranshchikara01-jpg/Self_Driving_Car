# Self_Driving_Car

📌 Overview

This project is a Computer Vision-based Self-Driving Car developed using Python, OpenCV, and YOLOv8. The system is capable of detecting road lanes, identifying surrounding vehicles and obstacles, estimating object distance, and making basic driving decisions based on the detected environment.

The project processes both recorded driving videos and live camera feeds to simulate the perception and decision-making modules of an autonomous vehicle.

🎯 Features

🚗 Lane Detection using OpenCV

🚦 Road Boundary Identification

🚙 Real-Time Object Detection using YOLOv8

📏 Distance Estimation for Detected Objects

↔️ Left / Right Lane Decision Making

🎥 Video-Based Simulation

📷 Live Camera Support

📊 Multi-Video Testing Support
